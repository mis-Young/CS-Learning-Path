# 2026.3.29 判断通过操作能否让字符串相等（2839）

## 题目描述

给你两个字符串 `s1` 和 `s2` ，两个字符串的长度都为 `4` ，且只包含 **小写** 英文字母。

你可以对两个字符串中的 **任意一个** 执行以下操作 **任意** 次：

- 选择两个下标 `i` 和 `j` 且满足 `j - i = 2` ，然后 **交换** 这个字符串中两个下标对应的字符。

如果你可以让字符串 `s1` 和 `s2` 相等，那么返回 `true` ，否则返回 `false` 。

## 解决方法

个人：注意到需要对比的只有字符串的奇偶两部分，且不需要关注顺序，因此我将字符串变成了列表，在列表里面嵌套两个列表，嵌套的列表即为奇和偶的字符串字母列表。然后再用`sort`函数进行排序，直接对比两个列表，若相等则符合题意返回true，否则返回false



更权威的解法类似，不过他的解法中将字符串的大小拓展到了n，不过对比奇偶是一样。还有就是其中划分字符串来对比并没有和我一样用列表，而是直接使用了字符串的切片并用了counter函数，一种可以判断字母是否同频的函数，更简洁
