# 2026.3.27 循环移位矩阵后的相似性检查

## 题目

输入m*n矩阵，和整数k

将矩阵中的奇数行右移k次，偶数行左移k次，判断操作后的矩阵与初始矩阵是否相同

## 解决方法

个人直接进行求解，复制初始矩阵后，判断奇偶行，各进行右移左移操作，最后进行对比判断



而官方答案更简洁，需要注意一点就是若一行左移k次与原来相同，那么右移k次也应该与原来相同。

因此，可以将各自左右移**统一为都是右移**。

然后在代码中不需要真的进行移动，因为只需要返回判断，因此**只用判断每一行中的第j个元素，与第(j + k)mod n个元素是否相同即可**。

若有不相同直接返回false，而遍历完所有元素没问题后就返回true。
